# MobileHub - Mobile Shopping Website Project

## 📱 Project Overview

**MobileHub** is a complete mobile shopping e-commerce website built with **HTML, CSS, and basic web concepts**. It's a beginner-friendly project designed to practice:

- HTML Lists & Navigation
- HTML Forms & Input Elements
- HTML Tables for Data Display
- HTML Multimedia (Images)
- CSS Styling & Layout
- CSS Responsive Design
- CSS Flexbox & Grid

---

## 📁 Project Structure

```
mobilehub-project/
│
├── index.html           # Home page
├── login.html           # Login page
├── register.html        # Registration/Sign-up page
├── products.html        # Product listing page
├── product.html         # Individual product details page
├── brands.html          # Mobile brands page
├── about.html           # About us page
├── contact.html         # Contact us page
│
└── style.css            # Main stylesheet for all pages
```

## 🚀 How to Use This Project

### Setup Instructions:
1. **Create a project folder** named `mobilehub` on your computer
2. **Save all HTML files** in the folder:
   - index.html
   - login.html
   - register.html
   - products.html
   - product.html
   - brands.html
   - about.html
   - contact.html

3. **Save the CSS file** as `style.css` in the same folder

4. **Open in Browser**:
   - Double-click `index.html` to open it in your browser
   - Use the navigation menu to visit different pages
   - All links are working and will navigate between pages

## 🐛 Common Issues & Solutions

**Issue**: Links not working
- **Solution**: Make sure all HTML files are in the same folder

**Issue**: Images not showing
- **Solution**: Check image URLs or use local image paths

**Issue**: CSS not applying
- **Solution**: Verify style.css is in the same folder and linked correctly

**Issue**: Responsive design not working
- **Solution**: Add `<meta name="viewport">` tag in HTML head (already included)
